# ============================================================================
# 하이퍼파라미터 조정 팁
# ============================================================================

**Contamination (오염률) - 중요도: ⭐⭐⭐⭐⭐**
- 이상치의 예상 비율 설정
- 너무 높으면: 많은 정상 데이터를 이상으로 판정
- 너무 낮으면: 이상치를 놓칠 수 있음
- 추천: 0.05 ~ 0.15 범위에서 시작

**n_clusters (KMeans) - 중요도: ⭐⭐⭐⭐**
- 엘보우 방법(Elbow Method)로 결정
- 너무 크면: 각 클러스터가 작아져 이상탐지 어려움
- 너무 작으면: 클러스터가 크니 이상치 분류 못할 수 있음
- 추천: 3 ~ 10 범위에서 시작

**n_components (PCA) - 중요도: ⭐⭐⭐⭐**
- 유지할 분산 비율 = explained_variance_ratio_의 합
- 0.9 = 90% 분산 유지
- 너무 높으면: 노이즈도 유지, 이상탐지 약함
- 너무 낮으면: 유용한 정보 손실
- 추천: 0.90 ~ 0.98 범위

**eps (DBSCAN) - 중요도: ⭐⭐⭐⭐⭐**
- 근방 거리 임계값
- k-distance graph로 "knee point" 찾기
- 추천: k=n_samples-1의 k-distance 그래프에서 knee point

# 메트릭 해석 가이드
# ============================================================================

| 메트릭 | 정의 | 해석 | 이상적 범위 |
|------|-----|------|-----------|
| **Precision** | 예측된 이상치 중 실제 이상치 비율 | 오탐율 추정 | > 0.8 |
| **Recall** | 실제 이상치 중 감지된 비율 | 미감지율(위험) | > 0.8 |
| **F1 Score** | Precision과 Recall의 조화평균 | 전체 성능 | > 0.8 |
| **ROC-AUC** | 다양한 임계값에서의 성능 | 모델 판별력 | > 0.85 |
| **TP/TN/FP/FN** | Confusion Matrix | 상세 분석 | - |

**성능 기준:**
- Excellent: F1 > 0.9
- Good: F1 > 0.8
- Acceptable: F1 > 0.7
- Poor: F1 < 0.7


# ============================================================================
# 문제 해결 체크리스트
# ============================================================================

[ ] 1. 낮은 Recall (놓친 이상치가 많음)
    → contamination 값 증가
    → 다른 모델 시도 (OneClassSVM, LocalOutlierFactor)
    → 데이터 정규화 확인

[ ] 2. 높은 False Positive (오탐이 많음)
    → contamination 값 감소
    → 모델 하이퍼파라미터 조정
    → 피처 엔지니어링 개선

[ ] 3. 모든 데이터가 이상/정상으로만 분류됨
    → 데이터 정규화 (StandardScaler, RobustScaler)
    → 피처의 스케일 확인
    → contamination 값 확인

[ ] 4. 메모리 부족
    → 배치 처리 사용
    → 데이터 샘플링
    → IsolationForest 또는 KMeansAnomaly 사용 (메모리 효율적)

[ ] 5. 학습 속도 느림
    → 더 빠른 모델 선택 (IsolationForest, KMeansAnomaly)
    → 데이터 샘플링
    → 배치 처리

**투표 방식 (Voting):**
```python
predictions = []
for model in [isolation_forest, pca_anomaly, kmeans_anomaly]:
    predictions.append(model.predict(X_test))

ensemble_pred = np.sign(np.sum(np.array(predictions), axis=0))
# 다수결로 이상/정상 판정
```

**점수 평균 (Score Averaging):**
```python
scores = []
for model in models:
    _, score = get_anomaly_predictions(model, X_test)
    scores.append(score)

ensemble_score = np.mean(scores, axis=0)
anomalies = ensemble_score > threshold
```

**가중 앙상블:**
```python
# 성능 좋은 모델에 더 높은 가중치
weights = [0.4, 0.35, 0.25]  # 모델 순서대로
ensemble_score = sum(w * s for w, s in zip(weights, scores))
```
